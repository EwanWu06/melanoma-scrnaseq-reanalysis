# Stage 3 Mini-Report: Cross-Method Comparison of Batch Integration on Tirosh 2016 Melanoma scRNA-seq

## 1. Abstract

This report presents Stage 3 Q3.4, a cross-method comparison of batch integration on 1,257 malignant cells from Tirosh et al. (2016, GSE72056). Two methods are compared: Harmony (linear post-PCA correction, Stage 2 baseline) and BBKNN (graph-based batch-balanced kNN, Q3.2). Stage 3 originally evaluated 6 candidate methods; 4 were ruled out across feasibility audit and runtime verification (scVI/scANVI, UCE: raw-count dependency; Seurat RPCA: R/Python interop instability; Scanorama: small-batch silent failure; scGen: upstream API drift and abandonment), each with an independent decision-log entry (a–f).

Both methods recover only two of Tsoi's four states (Melanocytic, Undifferentiated); neither recovers NGFR-dependent intermediate states (Transitory, Neural crest-like). Uniformly weak NGFR signal across all clusters in both methods establishes this as a dataset-level limitation rather than a method ceiling — a conclusion Stage 2's single-method analysis could not have reached. However, the two methods diverge structurally in how they handle batch-confounded cells: Harmony exposes batch artifacts in place, BBKNN absorbs them into biological clusters. The label "Undifferentiated" refers to 33 vs 635 cells across methods, with 2-cell overlap. The two methods solve different problems, not the same problem with different success rates.

## 2. Background

Melanoma malignant cells exhibit transcriptional heterogeneity along a dedifferentiation axis. Tsoi et al. (2018, *Cancer Cell*) defined four discrete states by four marker genes (MITF/SOX10/NGFR/AXL): **Melanocytic** (MITF↑ SOX10↑ NGFR↓ AXL↓), **Transitory** (adds NGFR↑), **Neural crest-like** (MITF↓ SOX10↑ NGFR↑ AXL↑), **Undifferentiated** (MITF↓ SOX10↓ NGFR↓ AXL↑). Balderson et al. (2024, *BFG*) recovered all four states from Tirosh 2016 malignant cells using a classical pipeline (PCA + Monocle trajectory inference) — but without batch correction, a methodological gap given the dataset's 19-patient composition and severely uneven per-patient cell counts (pt79 = 896 vs pt65 = 63, ~14× imbalance).

Stage 2 applied Harmony + Leiden clustering and recovered only 2 of 4 states (Mel + Undiff). The Stage 2 report (`docs/stage2_report.md`) attributed the Transitory/NC failure to two possible causes: dataset-level weak NGFR signal, and method-level residual batch effect at Harmony's default θ=2 (where the highest-NGFR cluster, cl3 with NGFR mean = 0.83, was dominated by patient pt80, preventing separation of biological signal from patient-specific effects). With n=1 method, the dataset-vs-method question was unresolvable.

Stage 3's scientific motivation is precisely this question: on the same data subset, apply a structurally different integration method and compare four-state recovery. If both methods fail at Transitory/NC through different mechanisms, the limitation is dataset-level; if failure modes differ, it is method-level. Raw-count-dependent methods are excluded due to DUOS access restrictions (entry a); Stage 3 is restricted to log-input methods.

## 3. Methods

Stage 3 initially planned four Core methods accepting log input (Harmony, Seurat RPCA, scGen, UCE), spanning three algorithm families (linear post-PCA, linear anchor-based, nonlinear VAE), with BBKNN (graph-based) as a Reach-tier extension outside these families. The candidate set converged to two methods across three verification rounds (`docs/decision_log.md` entries a–f, `docs/stage3_feasibility_audit.md`).

**Q3.1 paper-feasibility audit.** UCE was identified through source review as having a hidden raw-count dependency — its preprocessing applies `log1p(x/sum(x)*1000)` with a comment confirming counts are expected — and requires A100-class GPU (80 GB). Ruled out (entry b).

**Q3.2 implementation chain (entries c, d, e).** Seurat RPCA in R triggered `reticulate` autoprovisioning of pyenv Python 3.14, exceeded nbconvert timeout, and left orphan directories; an unstable R/Python bridge with no methodological return motivated pivoting to the family-equivalent Python-native Scanorama (entry c). Scanorama silently terminated after processing only two patients at default and reduced `dimred` settings — 6 of 15 patients have <30 cells, triggering per-batch SVD degeneracy (entry d). Pivoted to BBKNN (graph-based, no per-batch SVD), which converged with `approx=False` locked for reproducibility (entry e).

**Q3.3 scGen runtime verification (entry f).** PyPI scgen 2.1.0 failed to import (references `scvi._compat`, removed in scvi-tools 1.0+). GitHub master 2.1.1 passed import and training but crashed at `get_latent_representation()` and `batch_removal()` — scGen returns posterior tensors keyed `qz_m`/`qz_v` (scvi-tools 0.x convention) while scvi-tools 1.x expects `qz`/`qzm`/`qzv`. scGen has had no maintainer response for 21 months. API drift plus upstream abandonment is not patchable at the user level. Q3.1's paper-only audit missed this because it did not include runtime install + smoke test; that step has since been added as a mandatory audit criterion (entry f, "Lessons learned").

**Q3.4 comparison setup.** 1,257 malignant cells (15 patients, matching Balderson 2024 and Stage 2 scope); 30-D embeddings for both methods. Working resolutions: Harmony `leiden_r0.3` (6 clusters, Stage 2-selected), BBKNN `leiden_bbknn_r0.5` (3 clusters, Q3.2-selected). Metrics: **A1** patient silhouette on 2D UMAP (lower = better mixing); **A2** kNN patient purity in each method's native space (see §6 L2 for BBKNN graph saturation); **B1** per-cluster Tsoi marker expression (mean log2(TPM/10+1) + % expressing); **B2** Tsoi-state silhouette on 2D UMAP (higher = better state separation); **C1** cluster × Tsoi-state crosstab; **C2** cross-method partition agreement (ARI/NMI, strict Tsoi-state comparison).

## 4. Results

**Batch integration quality (A1/A2).** A1 patient silhouette on 2D UMAP: Harmony +0.256, BBKNN −0.031, raw baseline +0.578 (ΔHarmony = −0.32, ΔBBKNN = −0.61). BBKNN achieves more thorough patient mixing. A2 kNN purity in Harmony's 30-D integrated space: mean = 0.732 ± 0.33 (median 0.93); BBKNN graph A2 is structurally saturated (§6 L2) and not used as a quality comparator.

**Biological signal preservation (B1/B2).** B1 marker dotplot (`results/figures/07_combined_dotplot.png`, Visual 1) shows the bipolar structure is stable across methods — Melanocytic clusters (Harmony cl0/cl1, BBKNN 0) with MITF mean ≈ 4.3 and SOX10 ≈ 2.4, and the most-dedifferentiated clusters (Harmony cl4, BBKNN 1) with MITF dropping to 1.3–2.9 and AXL rising to 1.1–1.6. NGFR mean ≤ 0.83 and % expressing ≤ 25% across all clusters; no NGFR-high cluster emerges in either method. B2 silhouettes: Harmony self = 0.325, BBKNN self = 0.316; cross Harmony UMAP × BBKNN annotation = +0.292, BBKNN UMAP × Harmony annotation = +0.170.

**Cluster recovery and cross-method agreement (C1/C2).** Harmony at res=0.3 yields 6 clusters (Mel: cl0/cl1; Undiff: cl4; Other (batch?): cl2/cl3/cl5). BBKNN at res=0.5 yields 3 clusters (Mel: 0, Undiff: 1, Ambiguous: 2). C2 strict Tsoi-state agreement: ARI = +0.492, NMI = +0.395 (moderate).

**[P1] Both methods recover only Mel and Undiff; both fail at Transitory and Neural crest-like.** Intermediate states require elevated NGFR, which remains uniformly weak across all clusters in both methods (B1). Independent recovery of Stage 2's "2 of 4" finding under a structurally different integration mechanism **elevates this limitation from a Harmony-specific ceiling to a dataset-level constraint** — a conclusion Stage 2 alone could not have established.

**[P1b] The "2 of 4" framing partly reflects a discretization artifact.** B1 shows smooth marker gradients across clusters (MITF 4.3 → 1.3, AXL 0.5 → 1.6 inversely). Each method's third category — Harmony "Other (batch?)" and BBKNN "Ambiguous" — captures cells mid-gradient that resist assignment to either pole. The Tirosh malignant population appears continuum-dominated rather than discretely structured, consistent with Tirosh 2016's own preference for continuous signature scores over discrete state labels.

**[P2] The methods preserve biological signal at indistinguishable levels.** Self-silhouettes 0.325 vs 0.316 differ by 0.009, which is not interpretable at 2D UMAP precision as one method outperforming the other.

**[P3] Cross-method silhouette asymmetry reveals a mechanism-level difference.** Harmony UMAP × BBKNN annotation = +0.292 (high) vs BBKNN UMAP × Harmony annotation = +0.170 (low). The Harmony UMAP retains cluster structure even under BBKNN's labels because Harmony's residual patient lobes remain spatially separated; the BBKNN UMAP loses cluster structure under Harmony's labels because BBKNN has physically integrated cells Harmony marks as batch artifact. BBKNN's graph-level correction, propagated through UMAP, yields a substantively restructured 2D embedding; Harmony's PCA-level correction leaves patient lobes visible after UMAP (Visual 2, bottom panels).

**[P4] Partition-level agreement is moderate, but cell-level breakdown is more informative.** The Melanocytic core is stable: 374 cells are labeled Mel by both methods (92.6% of BBKNN Mel, 71.9% of Harmony Mel) — the most reproducible biological signal across methods. Of Harmony's 704 "Other (batch?)" cells, **81.4% (573) are absorbed by BBKNN into Undifferentiated**, 14.5% (102) into Ambiguous, 4.1% (29) into Melanocytic. BBKNN's Undifferentiated cluster is therefore dominated by cells Harmony marked as batch artifact.

**[P5] The label "Undifferentiated" refers to substantively different cell populations across methods.** Harmony's 33 Undiff cells overlap BBKNN's 635 Undiff cells by only 2 cells (0.3%). Harmony's 33 cells show extreme marker patterns (MITF mean 2.35, AXL 1.62, Stage 2 cl4) consistent with Tsoi's terminal Undifferentiated state. BBKNN's 635 cells show moderate marker patterns (MITF mean 2.90, AXL 1.10, BBKNN cl1) — the dedifferentiation trajectory's body, including most cells Harmony treated as batch. This sharpens an observation already on record from Stage 2 (`notebooks/03_harmony_umap.ipynb`, Observed caveats §B) that the malignant data look continuum-dominated, and is consistent with the Q3.2 annotation's trajectory framing (commit `fa29530`), which described BBKNN cl1 as the dedifferentiated end of the Melanocytic–Undifferentiated trajectory rather than a terminal state. Downstream analyses filtering on `tsoi_state == 'Undifferentiated'` extract entirely different cell sets across methods; cross-stage aggregation requires explicit reconciliation.

**[P6] Synthesis: the methods solve different problems.** Harmony **exposes batch artifact in place** — patient-dominated cells remain visible as patient-dominated clusters. Advantage: traceable patient origin. Cost: the dedifferentiation continuum's body fragments into multiple "batch artifact" clusters; their biological identity remains uncharacterized. BBKNN **absorbs batch artifact into biological structure** — patient-dominated cells are integrated into biological clusters revealing underlying dedifferentiation signal. Advantage: more cells assigned to biological states; the trajectory body is recovered as a coherent cluster. Cost: patient origin is lost as a structural feature; the 33-cell terminal Undifferentiated state loses its distinct identity, absorbed into the trajectory body. Choice depends on downstream goal: Harmony for batch detection and patient-aware analysis; BBKNN for maximum biological state recovery. **This is a design-level trade-off, not a ranking.**

## 5. Discussion

**Stage 3's contribution.** Stage 3 upgrades Stage 2's single-method finding into a two-mechanism comparison. The Transitory/NC failure is elevated from "Harmony ceiling" to dataset-level NGFR signal limitation — an upgrade impossible under n=1 method. A second finding emerges that is invisible under any single-method analysis: the same biological label refers to different cell populations across methods (P5). This finding's visibility derives precisely from cross-method comparison.

**Tirosh 2016 and Tsoi 2018.** Tsoi's four-state model was built on bulk RNA-seq from multiple cell lines, a setting with clear discrete structure. Tirosh 2016 itself preferred continuous signature scores along the same dedifferentiation axis. P1b supports the latter as a better fit for Tirosh single-cell data — the smooth MITF/AXL gradient suggests continuum-dominated structure where the four-state framework imposes imperfect discretization. This is not a conceptual failure of Tsoi's model but a context-specific mismatch with Tirosh-Smart-seq2-malignant data.

**Method-agnostic findings.** Three: (1) weak NGFR signal renders NGFR-dependent states marker-unrecoverable in this dataset under both mechanisms, transcending method choice; (2) the Melanocytic pole is the most reproducible biological signal (374 cells strongly consistent, P4), recommended as a high-confidence reference for downstream analyses; (3) same-named Tsoi states are not interchangeable across methods (P5), requiring explicit reconciliation for any cross-method or cross-stage aggregation.

**Methodology note.** This project's methodology incorporates real-time decision-log documentation, produced through iterative human-AI collaboration during exploration. Each of the 6 methods evaluated in Stage 3 has an independent decision log entry, including 4 ruled-out methods with distinct verification artifacts (smoke tests, source-code review, runtime tests). This documentation practice — uncommon in typical undergraduate portfolio projects — was made tractable by AI-assisted source verification and was iteratively refined by critical review of AI-suggested framings.

## 6. Limitations

**[L1] Silhouette precision and comparability.** All silhouettes (A1 patient and B2 Tsoi-state) are computed on each method's 2D UMAP. UMAP is a nonlinear, non-deterministic projection, and silhouette values inflate systematically relative to higher-dimensional embeddings (Stage 2 documented: Harmony silhouette = 0.080 on 30-D PCA vs 0.256 on 2D UMAP, ~3×). Within-report cross-method silhouette comparisons are directionally interpretable, but **absolute values should not be compared to Stage 2 or Q3.2 mini-report numbers, which used different conventions**.

**[L2] BBKNN A2 kNN purity is structurally saturated.** BBKNN's `neighbors_within_batch=3` × 15 patients = 45 neighbors per cell (44 after self-exclusion), of which exactly 2 are same-patient, so per-cell purity is fixed at 2/44 = 0.0455 with zero variance, independent of integration quality. Reported as a methodological observation about BBKNN's algorithm, not as a quality metric.

**[L3] ARI/NMI label-name invariance.** Cross-method ARI/NMI measure partition agreement, not label-name agreement. Collapsing the catch-all categories (Harmony "Other (batch?)" and BBKNN "Ambiguous") into a unified "Non-canonical" label was considered, but because ARI/NMI are insensitive to label names and each method's catch-all is already a single group, the collapse would not change agreement values. Only the strict comparison is reported.

**[L4] Single-resolution evaluation.** Both methods are evaluated at single working resolutions (Harmony res=0.3, BBKNN res=0.5), selected through independent annotation workflows in their respective stages. Q3.4 does not include a multi-resolution sweep — a natural Stage 4 extension (§7).

## 7. Conclusions and Future Directions

Stage 3 independently replicates Stage 2's "2 of 4 states recovered" result under a structurally different integration method (BBKNN), establishing the Transitory/NC failure as a dataset-level NGFR signal limitation rather than a Harmony-specific ceiling. A methodologically significant secondary finding — same-named Tsoi states refer to different cell populations across methods, with no winner-take-all between Harmony and BBKNN — emerges from cross-method comparison and is invisible under any single-method analysis. The originally planned 6-method comparison converged to 2 across three verification rounds (Q3.1 paper audit, Q3.2/Q3.3 runtime testing); each of the 4 ruled-out methods carries an independent verification artifact (decision log entries a–f).

Three Stage 4 candidates: (1) **cross-dataset robustness** — repeat the comparison on a complementary dataset with stronger NGFR signal (e.g., Jerby-Arnon 2018, GSE115978), directly testing P1's dataset-level conclusion; (2) **trajectory inference** — characterize the dedifferentiation continuum directly using Slingshot or scVelo (the latter requires raw counts, subject to DUOS constraints) rather than approximating with discrete state clusters, addressing P1b's continuum observation; (3) **multi-resolution sweep** — test whether P5's label divergence is robust to working-resolution choice, addressing L4.
